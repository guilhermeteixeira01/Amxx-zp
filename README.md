# CS 1.6 Zombie - Plugins AMXX

Este repositório contém plugins, addons e códigos fonte para servidores de **Counter-Strike 1.6**, com foco no **modo Zumbi (Zombie Mod)** e variações.

⚠️ **Avisos importantes**:
- Alguns plugins neste repositório são **modificados ou reaproveitados** de projetos open source de outros autores.
- **Vários plugins foram descontinuados** e podem conter **bugs** ou comportamentos inesperados.
- **Nenhum suporte oficial será fornecido** para correções ou dúvidas sobre os códigos.
- O uso é **por sua conta e risco**.

## 📁 Conteúdo
- Plugins AMXX personalizados
- Scripts SMA editáveis
- Configurações otimizadas para servidores Zumbi
- Recursos adicionais (sons, modelos, configs)

## 🎯 Objetivo
Manter uma biblioteca de plugins úteis para servidores CS 1.6, especialmente no modo Zombie, promovendo o reaproveitamento de conteúdo da comunidade.

## 🤝 Créditos
Damos crédito a todos os autores de código original sempre que identificados. Se você identificar conteúdo seu aqui sem menção, entre em contato.
